IP Authorized Circulation Sell Project

TIME：2018-11-11 Auth：5SJapan Team

　　Our project is to make a complete IP copyright operation ecosystem。

We also customize a side chain for each IP through the elastos support for the sidechain. IP currency, character ID authentication, copyright contract authorization, automatic sales sharing, copyright check and other functions are implemented on the side chain!

Using our website and APP, we can reduce the cost of copyright circulation in the real world, realize IP owners, IP purchase users, and end users ultimately benefit。

鉴于亦来云项目以太侧链的开发还未完成，我们的开发准备先在以太坊上实现发币和智能合约调用，通过网站和APP调用亦来云IDchain接口，实现版权授权的管理。

IP指“Intellectual Property”，直译为知识产权，原意为基于智力的创造性活动所产生的权利。

copyright的流通，是IP需求者（中间商）向IP拥有者购买授权，再依此授权进行二次商业开发的过程。消费者对中间商制作的goods进行消费，完成商业闭环。

痛点： 在传统的商业授权中，以动画商品授权为例，一次授权的时间成本为3个月，依照授权使用期限，授权资金成本在300～800万日元（最低分成金额）保底金+4%～10%项目分成（根据最低出货数制定授权金比率）该数据为一般商品化数据，包括T恤、手办等。 授权判定的依据则是 出货量高低 企画提交早晚 合作历史长短等 产权转移的确权在现有商业环境中是孤立的，流通迟滞。

解决方案： 使用ELA，为每一个IP生成一条侧链，在同一个IP生态内流通该IP的token。 通过亦来云的IDchain，为每一个商品赋予独立的编号，保证其正规性和稀缺性。使数字资产可以完成转移。

智能合约 将商品流通过程中产生的利益分配比例在最初写入智能合约，使参与各方可以分配得相应的利润。

参与方： IP持有方 IP中间商（购买产权、制作&贩卖商品） IP消费者

中间商向IP持有方购买商品化权，制作相关商品，贩卖给消费者。
